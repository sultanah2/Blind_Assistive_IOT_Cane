# Blind_Assistive_IOT_Cane

## 1. Project Overview & Abstract
Independent mobility is one of the major daily challenges faced by visually impaired individuals, especially in unfamiliar or dynamic environments[span_2](start_span)[span_2](end_span). Traditional white canes help users detect immediate, low-level obstacles but completely lack advanced features for communication, remote safety, and real-time navigation[span_3](start_span)[span_3](end_span). 

To solve these critical issues, this project introduces an integrated **Smart Cane System**—an embedded IoT solution designed to elevate the safety, independence, and confidence of visually impaired users[span_4](start_span)[span_4](end_span). 

The system operates on two core pillars[span_5](start_span)[span_5](end_span):
1. Obstacle Detection & Proximity Alert: Utilizing ultrasonic sensors to gauge distance from objects and triggering a buzzer with multi-level frequency beeps to alert the user dynamically[span_6](start_span)[span_6](end_span).
2. Emergency Emergency Alert System (Wireless Communication): Integrating a GPS module for precise location tracking paired with an HC-05 Bluetooth module[span_7](start_span)[span_7](end_span). When an emergency push-button is pressed, the cane instantly extracts coordinates, generates a Google Maps link, and transmits a "HELP!" message directly to a companion smartphone[span_8](start_span)[span_8](end_span).

## 2. Project Objectives
* Design and build an affordable, responsive smart cane to assist visually impaired individuals[span_9](start_span)[span_9](end_span).
* Enable active, real-time obstacle detection using ultrasonic waves[span_10](start_span)[span_10](end_span).
* Provide clear, varying auditory alerts (sound frequencies) based on obstacle distance[span_11](start_span)[span_11](end_span).
* Track and extract precise user coordinates (Latitude and Longitude) using GPS[span_12](start_span)[span_12](end_span).
* Deliver instant emergency SMS/messages with Google Maps integration via Bluetooth communication[span_13](start_span)[span_13](end_span).
* Enhance user autonomy, safety, and daily mobility[span_14](start_span)[span_14](end_span).
